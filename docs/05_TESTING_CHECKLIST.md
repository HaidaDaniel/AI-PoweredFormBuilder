# Manual Testing Checklist

Admin:
- Can login
- Can create form
- Can add all 3 field types
- Can edit field settings
- Can reorder fields
- Can delete field
- Can delete form

Public:
- Form renders correctly
- Required validation works
- Confirmation modal shows values
- Success message appears

AI:
- Adds valid field
- Rejects invalid type
- Does not corrupt form structure
